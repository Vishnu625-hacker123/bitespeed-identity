# Bitespeed Identity Reconciliation Service

## 📌 Overview
This service implements identity reconciliation logic using Node.js, Express, Prisma, and PostgreSQL.

It consolidates multiple contacts belonging to the same user based on shared email or phone number.

---

## 🚀 Tech Stack

- Node.js
- Express
- Prisma ORM
- PostgreSQL
- TypeScript
